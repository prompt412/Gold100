## 진행 방식

- 파일 이름: `2022_05_01_문제번호.cpp`
- `#문제풀이인증` 채널에 문제를 풀었다 자랑한다
- 일요일 9시가 되면 경아나 성민이 스터디원이 목표를 달성했는지 체크한다
